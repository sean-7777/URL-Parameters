URL Parameters

What are URL Parameters?

URL Parameters are also known as query strings or URL variables. They are something like this, your-website.com/welcome?fname=John&lname=Doe&age=23

How do you put it in a URL?

It’s simple. Just follow these rules

  1. First, decide if you want to put it on a homepage (domain.com, sdomain.domain.com, etc) or a subpage (domain.com/subpage). If on a homepage just put a slash. If you want it to be on a subpage then don’t put anything.
  2. Put a & after the URL to start the URL parameters section.
  3. For each URL Parameter you simply put in the parameter name, then a equals sign, and last, the parameter value. If you want to do multiple parameters, put an & in between.
  4. Here are some examples.
    1. site.com/?hello=True&something-else=i7g8
    2. site.sitte.com/?no=no
    3. site.com/page?chrachters=692

How do you access these parameters?

First, before we get into this question, I want to make something clear: You do not need to create a parameter, you just need to put it in the URL and it is created. Now, back to the question. There a two ways to do this,

  1. I do not recommend this method.

    // The URL is site.com/page?welcome=yes
var parameter = window.location.search
// Parameter is equal to ?welcome=yes

  2. Now of course there is a better method like a api right? Well yes, it’s called URLSearchParams. To get started, put this line in your code.

    // up is U in URl and p parameter
var up = new URLSearchParams(windows.location.window);

    Then, below is a list of the most commonly used methods. (Assuming the URL is hello.com/welcome?post=1234&action=edit)

    1. has
      This method detects if the URL contains a parameter.
      var has = up.has('post'); // True
var password = up.has('password'); // False
    2. get
      Takes the value of a parameter.
      var action = urlParams.get("action"); //"edit"
    3. getall
      Same as get instead in full.
      urlParams.getAll('action'); // ["edit"]
    4. tostring
      The full parameter section as in the URL.
      urlParams.toString(); // "?post=1234&action=edit"
    5. append
      Adds a parameter to the parameter section
      urlParams.append('active', '1'); // "?post=1234&action=edit&active=1"

  3. The URLSearchParams api is ideal, but not all browsers support it. So if you want to make it have more compatibility, then you can use pure Js.
    The function is the following.

    function getUrlParameterByName(name) {
    name = name.replace(/[\[]/, '\\[').replace(/[\]]/, '\\]');
	var regex = new RegExp('[\\?&]' + name + '=([^&#]*)');
	var results = regex.exec(location.search);
	return results === null ? '' : 
	decodeURIComponent(results[1].replace(/\+/g, ' '));
	};

    Here is an example. (The URL is the same as above, hello.com/welcome?post=1234&action=edit)

    getUrlParameterByName('post'); // "1234"
getUrlParameterByName('action'); // "edit"
